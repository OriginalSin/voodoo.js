Frequently Asked Questions
What browsers are supported?

Voodoo runs on all browsers that fully implement WebGL. For a complete listing, visit here.

Where are the demos?

There are several demos on the main page. Most likely your browser's WebGL support crashed. Try refreshing.

What tools are available for debugging?

The debug build of Voodoo writes trace messages to the console and performs asserts. There are also debug hooks in voodoo.debug. See the Documentation. Finally, the Support page has information about asking a specific question or reporting a bug.

Why did I get an error when accessing a variable in the View from the Model?

A Model may call any method on a View but must not access its variables. This is because Model.view and Model.stencilView are not actually instances of your views. Instead, they are composite views that wrap a set of views in an object that shares the same method interface. Variables are not exposed.
