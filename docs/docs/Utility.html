convertCssColorToThreeJsColor(cssColor) → {THREE.Color}

Converts a CSS color to a Three.Js color.

Legal CSS colors are defined here: http://www.w3schools.com/cssref/csscolorslegal.asp.

The alpha values of CSS colors are ignored.
