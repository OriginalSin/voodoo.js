Class: Engine

Engine

new Engine(opt_options)

Voodoo's main engine. It manages the renderer, the mouse detector, and all of the models. There can only be one engine per page and the user is responsible for creating it. If the user does not create an engine and assign it to voodoo.engine, then one will be created automatically with default options when the first model is instantiated.

Parameters:
Name Type Description
opt_options Options | Object

Options for voodoo.

Methods

destroy()

Shuts down the engine and stops rendering. After calling this, all models are invalid.
