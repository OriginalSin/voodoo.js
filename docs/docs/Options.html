Class: Options

Options

new Options(opt_options)

General options for initializing the voodoo engine. Pass this to the Engine when it is being created.

Parameters:
Name Type Argument Description
opt_options Object <optional>

Options settings to set.

Members

aboveLayer :boolean

Whether to create the above canvas that renders Z > 0. The user may wish to disable this completely when they are sure there will be no 3D content above the page.

Default is true.

aboveZIndex :number

CSS Z Index style for the above canvas.

Default is 9999.

antialias :boolean

Whether anti-aliasing should be enabled or not.

Default is true.

belowLayer :boolean

Whether to create the below canvas that renders Z < 0. The user may wish to disable this completely when they are sure there will be no 3D content behind the page.

Default is true.

belowZIndex :number

CSS Z Index style for the below canvas.

Default is -1.

fov :number

Maximum camera field of view in degrees along either axis. If zero, then the camera is an orthographic camera.

Default is 30.

frameLoop :boolean

Whether to automatically render and update each frame.

Default is true. If this is false, then the user must call voodoo.engine.frame() to render and update.

performanceScaling :boolean

Whether to scale down the resolution of the canvases if the frames per second drops below 45 for more than 3 seconds.

Default is true.

realtime :boolean

Whether to render every time the page is scrolled or resized, creating a very consistent experience at the expense of performance, or to render only when requestAnimationFrame runs, which is less frequent. If true, this also performs mouse raycasts in real-time rather than only on updates.

Default is true.

renderer :Renderer

3D rendering engine to use.

Default is ThreeJs.

renderInterval :number

The number of milliseconds between render frames. For example, if this is 100, then 100ms/frame is 10 frames/second. If this is 0, then the engine will render at the maximum framerate possible. frameLoop must be set to true for this to take effect.

Default is 0.

seamLayer :boolean

Whether to connect the seam between the above and below layers when using antialiasing. This results in an additional render layer and may affect performance. If antialiasing is false, this has no effect.

Default is true.

seamZIndex :number

The z-index for the seam layer. This should always be higher than the above layer z-index.

Default is 10000.

standardLighting :boolean

Whether to create a standard white ambient light and white camera light.

Default is true. If this is false, no standard lights are created and the user is responsible for creating any.

stencils :boolean

Whether to enable the stencil layers and the use of stencil views.

Default is true.

updateInterval :number

The number of milliseconds between update frames. For example, if this is 100, then 100ms/frame is 10 frames/second. If this is 0, then the engine will update at the maximum framerate possible. The frameLoop must be true for this to take effect.

Default is 0.
