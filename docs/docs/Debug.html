Class: Debug

Debug

new Debug()

Stores flags to help the user debug and develop Models. These only work in the debug build. voodoo.debug is automatically instantiated.

Members

disableStencils :boolean

Whether to disable stencils completely.

Default is false.

drawStencils :boolean

Whether to draw the stencil scene instead of the regular scene.

Default is false.

showFps :boolean

Whether to show the frames per second in the top left corner of the screen.

Default is false.
